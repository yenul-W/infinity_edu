Section 2

Formula Reference

Click any topic to expand the formulas you'll need for that sheet.

Mean = Σx ÷ n Sum of all values divided by the count
Mean (freq table) = Σ(f × x) ÷ Σf Multiply each value by its frequency, sum, then divide by total frequency

Median — the middle value when data is ordered. For n values, the median is at position (n + 1) ÷ 2. If n is even, average the two middle values.

Mode — the most frequently occurring value. A dataset can have no mode, one mode (unimodal), or more than one mode (bimodal / multimodal).

When to use each

Mean — best for symmetric distributions with no outliers

Median — best when outliers are present or data is skewed

Mode — best for categorical data or finding the most common value

Range = Max − Min Affected by outliers
Population variance σ² = Σ(x − μ)² ÷ n
Population std dev σ = √(Σ(x − μ)² ÷ n) Same units as the data

A larger standard deviation means the data is more spread out. A smaller standard deviation means values are clustered close to the mean.

On your calculator, use the σn (population) function, not σn−1 (sample), unless told otherwise.

Order the data, then split into four equal quarters.

Q1 = median of the lower half
Q2 = median of the whole dataset
Q3 = median of the upper half
IQR = Q3 − Q1 Interquartile range — middle 50% of data

Outlier fences (1.5 × IQR rule):

Lower fence = Q1 − 1.5 × IQR
Upper fence = Q3 + 1.5 × IQR

Any value below the lower fence or above the upper fence is classified as an outlier.

The five-number summary is: Min, Q1, Q2 (Median), Q3, Max.

Class width = Upper boundary − Lower boundary
Class midpoint = (Lower + Upper) ÷ 2 Used as the x value when calculating mean from grouped data
Relative frequency = f ÷ Σf
Cumulative frequency = running total of frequencies

In a histogram, bars are adjacent (no gaps) and bar width represents the class interval. Bar height (or area) represents frequency.

Distribution shapes: symmetric (bell-shaped), positively skewed (tail to the right), negatively skewed (tail to the left).

Type Subtype Example
Numerical Discrete Number of siblings, goals scored
Numerical Continuous Height, temperature, time
Categorical Nominal Eye colour, sport played (no natural order)
Categorical Ordinal Survey rating (strongly agree → strongly disagree)

Discrete — can only take specific values (usually countable whole numbers). Continuous — can take any value within a range (measured, not counted).
